## 2019.6.8 更新第一个项目 [TeachInCN](https://github.com/hy59/py4interesting/tree/master/TeachInCN) 🎓🎓🎓
本项目主要爬取外籍人员招聘网站JobLEADChina上的英语老师招聘信息1044条，万行教师人才网上的英语老师招聘信息8065条，并在此基础上对外教的招聘情况进行数据分析
